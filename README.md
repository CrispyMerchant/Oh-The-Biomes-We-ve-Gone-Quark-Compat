**Oh The Biomes We've Gone + Quark Compatibility**  

**Version:** 1.0  
**Minecraft:** Forge 1.20.1  

**Description:**  
This is the first release of the datapack for **Minecraft Forge 1.20.1**. It adds **compatibility between *Oh The Biomes We've Gone!* and *Quark***, allowing players to craft **chests using the modded woods** from *Oh The Biomes We've Gone!* in addition to the standard Quark chest recipes.  

**Features:**  
- Craft chests from **modded woods** provided by *Oh The Biomes We've Gone!*  
- Works seamlessly with **Quark chest recipes**  
- Fully compatible with **Forge 1.20.1**
